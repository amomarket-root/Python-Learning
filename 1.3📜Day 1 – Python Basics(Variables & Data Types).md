# üî¢ Variables \& Data Types ‚Äì Extended Examples

## 1. üì¶ Integer (`int`)

Whole numbers without a decimal point.

```python

# Number of products in stock

# Population of a city

```

## 2. üåä Floating Point (`float`)

‚ñ∂Ô∏è **Run Online**

## 3. üìù String (`str`)

Sequence of characters inside quotes (single `'`, double `"`, or triple `"""`).

```python

a multi-line

```

‚ñ∂Ô∏è **Run Online**

**Real-world Use:** Usernames, messages, data labels, file paths.

## 4. ‚úÖ Boolean (`bool`)

‚ñ∂Ô∏è **Run Online**

**Real-world Use:** Store status, on/off switches, verification results.

## 6. üîÑ Reassigning Variables and Dynamic Typing

Python allows variables to change type.

```python

print(type(x))

x = 10.5      # float
print(type(x))

x = "Ten"     # str
print(type(x))
```

‚ñ∂Ô∏è **Run Online**

**Use Case:** Adaptable scripting in dynamic systems.
